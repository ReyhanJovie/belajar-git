# Belajar GIT
Ini adalah deskripsi dari GIT


## Command Dasar
- Untuk liat graph, bisa dilakukan 
```bash
$ alias graph="git log --all --decorate --online --graph"
```
- untuk melihat branch yang sudah di buat
```bash
$ git branch
```
- untuk melihat branch yang sudah di merged
```bash
$ git branch --merged
```
